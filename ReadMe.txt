Para realizar a instalação dos imports e a criação do projeto react foi utilizado o pacote Yarn. Por isso,
é necessário que se tenha ele instalado, uma vez que será necessário baixar os node_modules. Além disso, 
para compilar o código é recomendado se utilizar o comando: yarn dev

O funcionamento do software estará explicado no PDF "Calculadora.pdf" .

As escolhas de implementação podem ser divididas em três:

    - Popups:
        Para a realização dos popups foi utilizada a biblioteca Sweet Alert 2, ela foi escolhida por conta
    de sua fama e também por ser de meu conhecimento prévio.

    - Serviço REST / Requisição get:
        Para a realização da requisição get com a API solicitada foi utilizada a biblioteca axios, ela foi
    escolhida por conta da sua fama dentro do react.

    - Estilização:
        Para a estilização do site foram utilizados o CSS e o seu framework Material-UI, essa escolha se deu
    por conta da facilidade no aproveitamento de ambos os recursos.
